# SoccerStat

Frontend-приложение для просмотра футбольной статистики.

## Технологии

- Vite
- React
- TypeScript

## Быстрый старт

```bash
npm install
cp .env.example .env
npm run dev
```

## Переменные окружения

- `VITE_API_TOKEN` — токен для запросов к API football-data.org.

## Доступные скрипты

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
